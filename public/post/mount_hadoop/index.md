## 安装java环境
安装jdk
设置java环境变量
>vim /etc/profile

```
export  JAVA_HOME=/usr/lib/jvm/java-7-openjdk-amd64/
eport   JRE_HOME=$JAVA_HOME/jre
eport   CLASSPATH=$JAVA_HOME/lib:$JRE_HOME:lib:$CLASPATH
export  PATH=$JAVA_HOME/bin:$JRE_HOME/bin:$PATH
```
